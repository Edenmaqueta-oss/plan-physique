# Plan Physique — PWA

Application mobile installable (PWA) : Protéines · Suivi IA · Mâchoire Carrée · Stats & Progression

## Installation rapide

```bash
npm install
npm run build
```

Puis dépose le dossier `dist/` sur **netlify.com/drop** (drag & drop).

## Installer sur iPhone
1. Ouvre le lien dans **Safari**
2. Icône Partager → **Ajouter à l'écran d'accueil**

## Installer sur Android
1. Ouvre dans **Chrome**
2. Menu ⋮ → **Ajouter à l'écran d'accueil**

## Clé API
Va dans l'onglet **Stats → Réglages** pour saisir ta clé API Anthropic (console.anthropic.com).
